# Customer Churn Prediction Web App

This project is a Machine Learning-based web application that predicts whether a customer is likely to churn or stay.

## 🚀 Features
- Predict customer churn using ML models
- Real-time predictions via Flask web app
- User-friendly interface
- Handles categorical data using encoders

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- HTML/CSS

## 📌 How it works
1. User enters customer details
2. Data is processed and encoded
3. Model predicts churn
4. Result is displayed instantly
